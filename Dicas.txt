Para adicionar o arquivo editado no repositório:

 - git add <FILE>;
 - git add --all;

Para commitar: 

 - git commit -m "Mensagem";

Para ver o status da pasta:

 - git status;

Para desfazer um add:

 - git reset;

Para desfazer um commit:

 - git reset HEAD~;

Trabalhe sempre no ramo de desenvolvimento, e depois espelhe as mudanças para o ramo principal.
Para mudar de ramos:

 - git checkout <ramo>;
